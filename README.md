# HeartRateLimits

## About
This project is a weekly assignment (Assignment 1) for the course "Mobile programming with Native Technologies". The application calculates heart rate limits based on user input and demonstrates basic usage of Jetpack Compose in Android development.

## Features
- Enter your age to calculate recommended heart rate limits.
- Simple UI built with Jetpack Compose.

## Usage
1. Run the app in an Android emulator or device.
2. Enter your age in the provided field.
3. The app displays your recommended heart rate limits.
